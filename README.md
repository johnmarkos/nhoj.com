# nhoj.com

Personal website for [John Markos O'Neill](https://nhoj.com). Static site deployed to GitHub Pages.

## What's here

- **[`my-code/`](my-code/)** — Tools and toys written by John
- **[`ai-code/`](ai-code/)** — Tools and toys built with AI assistance (includes an [auction app](ai-code/auction/), system design practice tool, and more)
- **[`pages/`](pages/)** — Standalone pages (reports, articles)

## Tech stack

No build system, no bundler, no frameworks. Pure HTML, CSS, and vanilla JavaScript. Deployed automatically from `main` via GitHub Pages.

Design uses a terminal/monospace aesthetic (JetBrains Mono) with automatic light/dark mode.
